 ## Usage 

  

 You can use [PicGo](https://github.com/Molunerfinn/PicGo) upload your images. 

  

 Set folder `/img`, repo `cdn`. 

  

 or 

  

 Click `update.sh` 

  

 ## Other Config 

  

 ### git lfs 

  

 Firstly, install [Git Large File Storage](https://git-lfs.github.com/). 

  

 Then: 

  

 ```sh 

 # in git repo 

 git lfs install 

 git lfs track "*.psd" 

 git add .gitattributes 

 ```
